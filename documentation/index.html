How to change a “Widescreen” to a “Boxed” style?

If you want to change the layout of your website from “Widescreen” to “Boxed” style, just go to the HTML (code) file of that page. You will see a division with a “theme-layout” class, i.e. <div class=”theme-layout”>. Here you need to add the class “boxed” in this way: <div class=”theme-layout boxed”>. Your page will be converted to “Boxed” style. For switching back to the “Widescreen” style, follow the same procedure and remove the class “boxed” from the div, i.e. <div class=”theme-layout”>.

How to change color scheme of your website template?

If you want to change the color scheme of any page, it is very simple. Choose a color of your choice among the ten (10) available color schemes that you can see in the CSS folder. Now go to the HTML coding of that page where you will see a code line as shown below:

<link rel="stylesheet" type="text/css" href="css/sea-green.css" />

For example, if you want to replace the sea green with orange, just replace the word “sea-green” with orange both in the css file and title as given in the code above

How to change header style to “Sticky Header”?

If you want to apply Sticky Header style for header on any page, go to the HTML code file and find the <header> tag. Add class=“sticky” in this tag as shown below

<header class=“sticky”>.

The header style will be converted to “Sticky Header” on the given page.

How to change header style to “Menu below Slider”? or How to show header below slider?

Open the HTML file of the page for which you want to show the header below the slider (LayerSlider). The next step is to cut the entire coding of the <header> tag and paste the same at the end of all the slides, but before the end of the division “layerslider-container-fw”. Now go to the <header> tag and add class=“below” in this tag as shown below:

<header class=“below”>.

The header style will be converted to “Menu below Slider” on the given page.

How to change header style to “With Logo in the Mid”?

For changing the header style to “With Logo in the Mid”, just go to the <header> tag in the code file of the respective file and add class=“middle-logo” in this tag. That is all.

How to change header style to “Toggle Header”?

In the HTML file of any page where you want to show “Toggle Header”, go to the <header> tag and add class=“toggle-header”. The next step is to go to the division with id=“top-bar” and add the class=“top-bar-toggle”. In the third step, add the following line above the same division with id=“top-bar”:

<div class="show-header"><i class="icon-reorder"></i></div>
